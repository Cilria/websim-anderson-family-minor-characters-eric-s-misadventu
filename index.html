Niobium Linux 1.0 Released!

Release Posted by u/elemental_dev • 45 minutes ago

It's here! After 8 months of development since our initial announcement, Niobium Linux 1.0 is now available for download. Launch went smoothly and we're excited to see what you think.

What's New in 1.0

  • ✅ Stable release based on Debian 8.7
  • ✅ Solar Desktop Environment 1.0 (KDE-based)
  • ✅ Commercial package repository ready
  • ✅ 24/7 enterprise support now active
  • ✅ Official documentation complete

Download & Pricing

  • Evaluation ISO: Free download, 30-day trial
  • Desktop License: $99/year per workstation
  • Server License: $299/year per server
  • Enterprise: Custom pricing available

Download Niobium Linux 1.0

ISO images available for x86_64 and ARM64. Includes live environment and installer. MD5 checksums provided.

# Quick install for evaluation wget https://elementalsoft.com/downloads/niobium-1.0-x86_64.iso sha256sum niobium-1.0-x86_64.iso # then burn to USB or use in VM

Thanks to everyone who participated in the beta program! Special shoutout to the enterprise testers who helped us refine the commercial features.
